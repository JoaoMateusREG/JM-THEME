## Installation

1. Install theme from the [Marketplace](https://marketplace.visualstudio.com/items?itemName=JoaoMateusREG.jm-dark-theme)
2. Go to **File > Preferences > Color Theme** (ou `Ctrl+K Ctrl+T`)
3. Select **JM Dark Theme**
